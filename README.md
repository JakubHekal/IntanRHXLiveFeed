# Intan RHX Live Feed

Stream, visualize, and analyze neural recordings from Intan RHX systems in real-time.

## Quick Start

### Requirements

- Python 3.8+
- Windows (Intan RHX TCP interface)

### Installation

```bash
pip install -r requirements.txt
```

### Usage

```bash
python main.py
```
